Om siden:

Denne siden er min første fullskjermsapplikasjon, og er kun laget for å få øving i HTML, CSS og Javascript. Den blir også levert som min første obligatoriske oppgave på Start IT.

Obligatorisk oppgave 1:

Hva man lager er helt fritt, men det skal lages en enkel applikasjon som inneholder alt som er listet opp under. Og alt skal brukes til noe meningsfullt. Du må finne på noe selv - det kan ikke være en av oppgavene i kurset. (Men det kan være noe som ligner.)


  • Fullskjermslayout med CSS Grid - og meny knapp som skjuler/viser en meny med minst to valgmuligheter
  • Minst en global variabel
  • Minst en funksjon - og minst en onclick eller annen event
  • Minst en if-setning
  • Bruk av nøkkelordet this
  • Applikasjonen skal endre på HTML-dokumentet, ved hjelp av InnerHTML
